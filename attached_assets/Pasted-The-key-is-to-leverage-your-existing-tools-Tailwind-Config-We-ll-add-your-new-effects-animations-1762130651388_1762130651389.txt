The key is to leverage your existing tools:

Tailwind Config: We'll add your new effects (animations, gradients) to your tailwind.config.js file as custom design tokens.

next-themes Hook: We'll use the useTheme hook to conditionally render the light-mode-only effects (Paper Texture, Light Rays).

GSAP: We will use GSAP (which you already have) for the "Ambient Light Rays" instead of CSS animations. It's more powerful and controllable.

Shadcn: We can apply these new styles directly to your Shadcn components (<Card>, <Badge>) using the className prop.

## 1. Living Ink & Watercolor Gradients (The Tailwind Way)
Instead of a separate CSS file, we'll add this directly to your theme.

1. In your tailwind.config.js file:

JavaScript

/** @type {import('tailwindcss').Config} */
module.exports = {
  theme: {
    extend: {
      backgroundImage: {
        'watercolor-sunset': 'linear-gradient(-45deg, #FFC3A0, #FFAFBD, #E1BEE7, #D4A5A5)',
      },
      backgroundSize: {
        '400': '400% 400%',
      },
      keyframes: {
        watercolorShift: {
          '0%': { backgroundPosition: '0% 50%' },
          '50%': { backgroundPosition: '100% 50%' },
          '100%': { backgroundPosition: '0% 50%' },
        },
      },
      animation: {
        'watercolor-shift': 'watercolorShift 20s ease infinite',
      },
    },
  },
  plugins: [],
}
2. In your React Components (e.g., a Shadcn <Badge>):

Now you can just use these classes.

JavaScript

import { Badge } from "@/components/ui/badge";

<Badge className="bg-watercolor-sunset bg-400 animate-watercolor-shift text-white">
  60:1
</Badge>

## 4. & 5. Light Rays & Paper Texture (The next-themes Way)
This is the most elegant solution. We'll create one or two new components that only render if the theme is "light."

1. Create a new component (e.g., components/ThemeEffects.jsx):

JavaScript

import { useTheme } from 'next-themes';
import { useEffect, useRef } from 'react';
import { gsap } from 'gsap';

export function ThemeEffects() {
  const { resolvedTheme } = useTheme();
  const lightRayRef = useRef(null);

  // --- 4. Ambient Light Rays (GSAP) ---
  useEffect(() => {
    if (resolvedTheme === 'light' && lightRayRef.current) {
      // Animate the light ray
      gsap.to(lightRayRef.current, {
        x: '200vw', // Move it far across the screen
        opacity: 0,
        duration: 30,
        ease: 'linear',
        repeat: -1,
        delay: -15, // Start halfway through
      });
    }
  }, [resolvedTheme]);

  // Only render these effects if the theme is light
  if (resolvedTheme !== 'light') {
    return null;
  }

  return (
    <>
      {/* --- 5. Paper Texture Overlay --- */}
      {/* This SVG filter is invisible, just provides the 'noise' definition */}
      <svg style={{ position: 'absolute', width: 0, height: 0, overflow: 'hidden' }}>
        <filter id="paper-texture">
          <feTurbulence type="fractalNoise" baseFrequency="0.65" numOctaves="3" stitchTiles="stitch" />
        </filter>
      </svg>
      {/* The visible overlay div that uses the filter */}
      <div 
        className="fixed inset-0 z-[9999] opacity-[0.03] pointer-events-none"
        style={{ filter: 'url(#paper-texture)' }}
      />

      {/* --- 4. Ambient Light Rays (The Element) --- */}
      <div
        ref={lightRayRef}
        className="fixed top-0 left-[-100vw] w-1/2 h-screen 
                   bg-gradient-to-r from-transparent via-white/10 to-transparent 
                   -rotate-45 blur-3xl opacity-20 pointer-events-none z-[9998]"
      />
    </>
  );
}
2. In your main App.jsx or Layout.jsx file:

Render this component once at the top level, inside your ThemeProvider.

JavaScript

import { ThemeProvider } from "next-themes";
import { ThemeEffects } from "@/components/ThemeEffects";

function App() {
  return (
    <ThemeProvider attribute="class" defaultTheme="system" enableSystem>
      <ThemeEffects /> {/* <-- Add it here */}
      {/* ... The rest of your app ... */}
    </ThemeProvider>
  );
}
## 8. & 10. Soft Focus & Museum Lighting (Tailwind Utilities)
These are now extremely simple.

Soft Focus (Bokeh): On your sticky nav component, just add Tailwind's backdrop-blur utility.

JavaScript

// Example for your nav bar
<nav className="sticky top-0 z-50 bg-white/70 dark:bg-black/50 backdrop-blur-lg">
  ...
</nav>
Museum Gallery Lighting: We can add this as a custom gradient in tailwind.config.js just like the watercolor.

1. In tailwind.config.js (inside theme.extend.backgroundImage):

JavaScript

'gallery-light': 'radial-gradient(circle at 50% 40%, rgba(255, 255, 255, 0.08) 0%, transparent 70%)',
'gallery-dark': 'radial-gradient(circle at 50% 40%, rgba(255, 255, 255, 0.05) 0%, transparent 70%)',
2. In your Shadcn <Card> component:

JavaScript

import { Card } from "@/components/ui/card";

// Use a pseudo-element for the overlay
<Card className="relative overflow-hidden 
               before:content-[''] before:absolute before:inset-0 
               before:bg-gallery-light dark:before:bg-gallery-dark 
               before:pointer-events-none">
  {/* Card content... */}
</Card>
This approach integrates perfectly with your stack and respects your theme-switching logic.
